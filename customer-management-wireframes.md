## 전체 레이아웃 구조

### 페이지 기본 레이아웃
```
┌─────────────────────────────────────────────────────────────────────────────────────┐
│ Sidebar (256px) │ Header (PRO, 문자 잔여, 뒤로가기, 새로고침, 지점선택, 로그아웃)     │
│                 ├───────────────────────────────────────────────────────────────────│
│ [Navigation]    │ Main Content Area (padding: 24px)                              │
│                 │                                                                 │
│ • Home          │ ┌─ 페이지 제목 (text-3xl font-bold text-gray-800) ─────┐       │
│ • 고객관리       │ │ "상담 고객 관리" / "미등록 고객 관리" / "신규 등록 고객 관리"  │       │
│   - 상담 고객    │ └─────────────────────────────────────────────────────┘       │
│   - 미등록 고객  │                                                                 │
│   - 신규 등록    │ ┌─ 날짜 네비게이션 & 검색 & 액션 버튼 (flex justify-between) ─┐ │
│ • 메시지 관리    │ │ [◀] [2024-01-01 ~ 2024-12-31] [전체▼] [▶] [🔍 검색] [+추가] │ │
│ • 캠페인 관리    │ └─────────────────────────────────────────────────────────────┘ │
│ • 상품/결제관리  │                                                                 │
│ • 통계          │ ┌─ 테이블 영역 (bg-white shadow rounded) ────────────────────┐   │
│ • 설정          │ │ [Table Header - bg-gray-400 text-white]                    │   │
│                 │ │ [Table Body - striped rows]                                │   │
│                 │ │ [Action Buttons in each row]                               │   │
│                 │ └────────────────────────────────────────────────────────────┘   │
└─────────────────┴─────────────────────────────────────────────────────────────────┘
```

### 공통 스타일 가이드
- **배경색**: #f7f9fb (페이지), #ffffff (카드)
- **그림자**: shadow (카드)
- **간격**: space-y-6 (섹션 간), gap-4 (요소 간)
- **테두리**: border-gray-300, rounded
